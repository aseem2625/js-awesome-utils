A function to convert a string to title-cased string. Eg: 'hello_world_thiS_iS' to 'Hello World This Is'

stringToTitleCase(str: String, splitToken: String): String
Parameters
str (String = '') A string to be processed
splitToken (String = ' ') A token on basis of which str is splitted
Returns
String: A title-cased string
Example
'hello_world_thiS_iS' to 'Hello World This Is'
